# Emilia

Proyecto construido en Express que gira alrededor de crear el sitio web de una cantante.

### Etapa 1

- Estructura HTML.
- Estilos CSS.
- Diseño responsive.

### Etapa 2

- Migración a MVC.
- Uso de `Express-Generator`
- Uso de template engines (ejs) en lugar de HTML.
- Uso de partials.
- Manejo de rutas.
- Uso de controladores.
